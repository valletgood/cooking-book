# CookingBook Design Context

## Target Audience
- 요리를 좋아하는 개인 사용자
- 주방에서 스마트폰으로 사용 (모바일 웹 퍼스트)
- 소규모 (1~5명)

## Use Cases
- 레시피 URL/사진을 AI로 파싱하여 저장
- 저장된 레시피를 쿡 모드로 따라 요리
- 요리 기록 관리

## Brand Personality
- **톤**: 따뜻하고 친근한 오두막(cottage) 느낌
- **캐릭터**: 코코(Coco) — 요리사 모자를 쓴 토끼 캐릭터
- **색상**: 오렌지 primary, 크림/베이지 계열 배경
- **감성**: 아늑하고 포근한 주방, 손글씨 느낌, 나무/자연 소재감
- **테마**: 라이트 테마만 (다크 모드 없음)

## Tech Stack
- Next.js App Router + TypeScript + TailwindCSS v4 + shadcn/ui
- 모바일 웹 기준 (375px)
